# 📘 SPK Ders Notları - İnteraktif Sınav

Sermaye Piyasası Kanunu ve Mevzuatına hazırlık için hazırlanmış, **interaktif sınav uygulamasıdır.**

## 🎯 Özellikler

- ✅ **Cevap gizli** – Önce düşün, sonra cevabı gör!
- ✅ **Anında geri bildirim** – Doğru/yanlış ve açıklama
- ✅ **Detaylı konu anlatımı** – Kaydırmalı slaytlar
- ✅ **İleri/Geri/Rastgele** soru navigasyonu
- ✅ **Klavye desteği** (→ ileri, ← geri, R rastgele)
- ✅ **Tam mobil uyumlu** (Telefon, tablet, bilgisayar)

## 🚀 Hemen Dene

👉 [Uygulamaya Git](https://kullaniciadiniz.github.io/spk-sinav/)

## 📂 Dosya Yapısı

- `index.html` – Ana uygulama dosyası
- `data.json` – Tüm sorular, şıklar, açıklamalar ve slaytlar
- `README.md` – Proje tanıtımı

## 📝 Nasıl Yeni Soru Eklenir?

1. `data.json` dosyasını aç
2. `"questions"` listesine yeni bir soru ekle (yukarıdaki örnekleri takip et)
3. GitHub'a gönder, site otomatik güncellenir!

## 🛠️ Teknik Detaylar

- HTML + CSS + JavaScript (sadece istemci tarafı)
- Veri: JSON (kolay güncellenebilir)
- Hosting: GitHub Pages (ücretsiz)
